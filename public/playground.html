LLM Settings

Simulated Mode (Recommended): Fast, free, pattern-based testing. Perfect for learning about prompt injection attacks and testing security controls. No API key required.

Real LLM Mode (Advanced): Test against actual GPT-4 or Claude models for production validation. Requires API key and incurs costs (~$0.01-0.10 per test).

Ollama (Local): Test with local models running on your machine. Requires Ollama installed and running on localhost:11434. No API key or costs.

Production Validation Mode: Use this to test your prompts against the exact LLM you'll use in production.
Security: API key stored in browser localStorage only, never sent to DVAA servers.
Cost: Each test runs 9 attacks. Typical cost: $0.01-0.10 per full test depending on model.
Rate Limits: LLM providers may rate-limit or temporarily block API keys after multiple rapid tests. If your key stops working, wait 5-10 minutes or create a new key.
